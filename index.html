Latest Project

Inventory Management System


For my second group project in my UC Berkeley Coding Bootcamp cohort, my partner and I created an inventory management system for a fictitious Used Car Sales Lot. This web application incorporates technologies such as HTML, CSS, JavaScript, Bootstrap, Node.js, Express.js, Handlebars, and MySQL. We implemented CRUD operations for vehicle records, built API routes for GET and POST requests, and used MySQL queries for efficient database management. The project showcases our ability to create, retrieve, update, and delete vehicle records, providing a dynamic and user-friendly experience.

Movie Knight - Group Project


The Movie Knight project is an innovative web application that allows users to search for and explore detailed information about movies effortlessly. Utilizing server-side APIs, the user can search for a movie by its title, and the website will return the results from The Movie Database (TMDB) and IMDb APIs. Moreover, Movie Knight enables users to create and manage their personalized watchlist, with seamless integration of local storage for persistent data retention. The project exemplifies a professional and polished implementation, utilizing HTML, CSS, JavaScript, and various libraries, ensuring responsiveness and a captivating UI design.
